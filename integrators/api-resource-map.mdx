# Карта API-ресурсов

Эта страница помогает быстро понять, где в API живут основные бизнес-объекты платформы.

Для большинства интеграций важно сначала определить:

- нужную API-поверхность
- нужную группу ресурсов
- необходимость webhook-синхронизации

На этой странице интеграторам представлена ​​практическая карта основных групп ресурсов, предоставляемых One Link Cloud. Это не замена контрактам уровня запроса, но оно показывает, где находится каждая бизнес-возможность.

## Основные ресурсы для учетной записи

Основная рабочая поверхность API ограничена учетной записью.

## Группы ресурсов

| Группа ресурсов | Типичная форма пути | Цель |
| --- | --- | --- |
| Контакты | `/api/v1/accounts/:account_id/contacts` | Идентификация клиентов, поиск, фильтрация, импорт/экспорт |
| Беседы | `/api/v1/accounts/:account_id/conversations` | Потоки обмена сообщениями, назначение, метки, настраиваемые атрибуты |
| Сообщения | `/api/v1/accounts/:account_id/conversations/:conversation_id/messages` | Операции с входящими и исходящими сообщениями |
| Компании | `/api/v1/accounts/:account_id/companies` | Записи клиентов на уровне организации |
| Inbox-очереди | `/api/v1/accounts/:account_id/inbox-очереди` | Очереди и конфигурация каналов |
| Правила автоматизации | `/api/v1/accounts/:account_id/automation_rules` | Правила рабочего процесса, управляемого событиями |
| Macros | `/api/v1/accounts/:account_id/macros` | Многоразовые действия, запускаемые оператором |
| CRM | `/api/v1/accounts/:account_id/crm/...` | Воронкаы, этапы, сделки, задачи, определения полей |
| Планирование | `/api/v1/accounts/:account_id/scheduling/...` | Ресурсы, услуги, назначения, платежи, расходы |
| Captain | `/api/v1/accounts/:account_id/captain/...` | Помощники, документы, специальные инструменты, copilot |
| Интеграции | `/api/v1/accounts/:account_id/settings/integrations` и связанные ресурсы аккаунта | Жизненный цикл подключенного приложения и поведение при интеграции |
| Portal и статьи | `/api/v1/accounts/:account_id/portals/...` | Содержание и настройки базы знаний |

## CRM Области ресурсов

CRM сгруппирован под `/crm` и в настоящее время включает в себя:

- воронки
- этапы
- статусы задач
- определения полей
- сделки
- задачи
- комментарии и сроки в рамках подресурсов сделок и задач.

## Планирование областей ресурсов

Планирование сгруппировано под `/scheduling` и в настоящее время включает в себя:

- календарь
- ресурсы
- контакты
- услуги
- встречи
- оплата встреч
- платежи и расходы
- праздники
- переопределения рабочего дня
- выходные

## Captain Области ресурсов

Captain сгруппирован под `/captain` и в настоящее время включает в себя:

- помощники
- Крепления inbox
- сценарии
- документы
- ответы помощника
- Темы и сообщения copilot
- специальные инструменты
- задачи редактора, такие как переписывание, подведение итогов, предложение ответа и создание последующих действий.

## Client API Области ресурсов

Для клиентов чата, работающих с клиентами, поверхность виджета/клиента включает в себя:

- конфигурация виджета
- диалоги с клиентами
- сообщения клиента
- обновления контактов с клиентами
- этикетки
- прямые загрузки

## Как использовать эту карту

1. Сначала выберите правильную поверхность API.
2. Найдите группу ресурсов, ближайшую к вашему бизнес-объекту.
3. Используйте webhooks, если вам нужна синхронизация event-driven вместо опроса.
4. Используйте сетевой поток пользовательского интерфейса в качестве окончательного ориентира, когда поведение должно точно соответствовать продукту.
