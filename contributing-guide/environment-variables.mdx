# Environment Variables for Development

This guide covers environment variables specifically for development and testing in the Onelink fork.

For production environment variables, see the [Self-hosted Environment Variables](/self-hosted/configuration/environment-variables) guide, which remains part of the inherited operational deployment base.

### Use letter opener instead of mailhog/SMTP

Set the following variable to open emails in letter opener instead of SMTP

```bash
LETTER_OPENER=true
```
