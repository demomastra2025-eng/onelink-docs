# Construction Profile

The construction profile should extend shared platform behavior with project-specific needs such as:

- object/project lifecycle
- site and address context
- estimation and budget fields
- milestone-oriented workflows
- project dashboards and construction-specific reports

The construction profile should stay isolated from healthcare-specific logic and reuse shared CRM and support mechanics whenever possible.
