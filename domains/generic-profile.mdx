# Generic Profile

`generic` is a first-class domain profile, not an undefined fallback.

It should provide:

- standard CRM behavior
- configurable pipelines and stages
- shared fields plus tenant custom attributes
- general-purpose dashboards and reports
- no hard dependency on healthcare or construction vocabulary

Use `generic` when a customer needs broad CRM/support capabilities without a strict vertical workflow.
